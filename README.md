# 🐍 Python Basics for Machine Learning

A structured collection of Jupyter notebooks covering core Python concepts — from variables and data types all the way to OOP and functional programming. Designed as a solid foundation before diving into data science and machine learning libraries.

---

## 📚 Notebooks

| # | Notebook | Topics Covered |
|---|----------|----------------|
| 01 | `01_python_basics_variables_and_datatypes.ipynb` | `print()`, variables, naming rules, data types (`int`, `float`, `str`, `bool`), arithmetic operators |
| 02 | `02_python_typecasting_and_data_structures.ipynb` | Type casting (`int()`, `float()`, `str()`), `input()`, Lists, Tuples, Sets, Dictionaries |
| 03 | `03_python_lists_in_depth.ipynb` | Indexing, slicing, negative indexing, nested lists, `append()`, `extend()`, `insert()` |
| 04 | `04_python_string_methods.ipynb` | f-strings, `.format()`, multi-line strings, case methods, `split()`, `join()`, `strip()`, `replace()`, `isalpha()`, `isdigit()` |
| 05 | `05_python_list_methods_and_tuples.ipynb` | `pop()`, `remove()`, `clear()`, `sort()`, `sorted()`, `reverse()`, `copy()`, `count()`, `index()`, tuple unpacking |
| 06 | `06_assignment_conditional_statements.ipynb` | *(Assignment)* even/odd checker, positive/negative/zero, vowel checker, leap year, grade calculator, traffic light simulator |
| 07 | `07_python_conditionals_and_loops_intro.ipynb` | `if / elif / else`, `and` / `or`, nested `if`, `for` loop, `range()`, `continue`, `break`, `while` loop |
| 08 | `08_python_loops.ipynb` | `for` loop, `enumerate()`, `range()` tricks, `while` loop, Fibonacci, digit sum, star patterns |
| 09 | `09_python_functions.ipynb` | `def`, `return`, parameters, local vs global variables, keyword & default arguments, `*args`, `**kwargs`, prime checker |
| 10 | `10_python_functions_and_exception_handling.ipynb` | Advanced `*args` / `**kwargs`, `try / except / finally`, `ZeroDivisionError`, `TypeError`, `NameError` |
| 11 | `11_assignment_calculator_and_star_patterns.ipynb` | *(Assignment)* Menu-driven calculator with exception handling, nested loop star patterns (triangle, diamond) |
| 12 | `12_python_oop_classes_and_inheritance.ipynb` | Classes, `__init__`, class vs instance variables, `getattr` / `setattr` / `hasattr` / `delattr`, single/multilevel/multiple inheritance, `super()`, encapsulation, inner classes |
| 13 | `13_python_list_comprehension_map_filter_lambda.ipynb` | List comprehension, walrus operator `:=`, `filter()`, `map()`, `lambda`, flattening nested lists |

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Jupyter Notebook or JupyterLab

### Running locally

```bash
# Clone the repository
git clone https://github.com/mdtahmedulhassan/Python-basic-for-machine-learning.git
cd Python-basic-for-machine-learning

# Install Jupyter if you don't have it
pip install notebook

# Launch Jupyter
jupyter notebook
```

Then open any `.ipynb` file from the browser interface.


## 🧭 Recommended Learning Path

Follow the notebooks in order (01 → 13). Each one builds on the previous.

```
Variables & Types → Data Structures → Strings & Lists
    → Conditionals → Loops → Functions → OOP → Functional Python
```

Notebooks marked **Assignment** (06, 11) are hands-on practice — try solving them yourself before looking at the solutions.

## 🔑 Key Concepts at a Glance

- **Data types & casting** — working with `int`, `float`, `str`, `bool` and converting between them
- **Data structures** — Lists, Tuples, Sets, Dictionaries with all their built-in methods
- **Control flow** — `if/elif/else`, `for`, `while`, `break`, `continue`
- **Functions** — reusable code blocks, `*args`, `**kwargs`, scope, exception handling
- **OOP** — classes, inheritance, encapsulation using real examples
- **Functional tools** — list comprehension, `map()`, `filter()`, `lambda`
